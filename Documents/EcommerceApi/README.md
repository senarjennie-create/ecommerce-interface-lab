A RESTful API backend for an e-commerce product catalog built with Spring Boot.

## 🛠️ Technologies Used

- Java 25

- Gradle
- Spring Web
- Lombok
- Jakarta Validation

## 📋 Project Overview

This project implements a complete RESTful API for managing an e-commerce product catalog. It demonstrates understanding of HTTP methods, status codes, REST principles, and CRUD operations.

### Features

- Create, Read, Update, Delete (CRUD) operations for products
- In-memory data storage using `ArrayList<Product>`
- Product filtering by category, price, and stock quantity
- Proper HTTP status codes (200, 201, 204, 404, 400, 500)
- Global exception handling with consistent error responses
- Input validation for product fields
